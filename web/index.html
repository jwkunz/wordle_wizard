How To Use Wordle Wizard

This app is an external Wordle advisor. You play Wordle somewhere else, then use this page to record each guess and the feedback you received. The solver keeps track of the remaining valid answers and ranks good next guesses.

  1. Enter the exact 5-letter guess you used in the Current Guess box.
  2. Click each feedback tile until it matches the game result: grey for miss, yellow for present, green for correct.
  3. Press Apply Feedback. The history, candidate list, and recommendations update immediately.
  4. Repeat for each round until the candidate set is small enough or you solve the puzzle.

Reading The Main Panels

History shows every guess you have already entered. This is the fastest way to confirm the solver state matches your external game.

Remaining Candidates shows answers still consistent with all entered feedback.

Recommendations ranks next guesses by entropy. If you enable Candidate only, the list is restricted to words that are still possible answers. Leave it off if you want exploratory probe guesses.

Diagnostics exposes the current state in a more raw format for debugging and validation.

Dictionary Behavior

The bundled dictionary is always available. Remote dictionaries are optional.

  • When you fetch a remote word list, the app caches it in browser storage.
  • On refresh, the app now tries to restore the active cached remote dictionary automatically.
  • The cache manager reports every saved dictionary, when it was stored, and how many filtered 5-letter words it contains.
  • You can activate a cached dictionary, remove a single cached dictionary, or clear the cache entirely.

If a remote source contains mixed content, only unique lowercase 5-letter alphabetic words are kept.

Common Workflow

A good default workflow is:

  1. Open the workspace tab.
  2. Choose the bundled list or load a cached remote dictionary.
  3. Enter one guess at a time from your live game.
  4. Use the recommendations table to choose either the strongest information-gain probe or a remaining candidate.
  5. Press New Game when you start a new puzzle. New Game keeps the active dictionary and clears only the current puzzle state.

New Game vs Dictionary Changes

New Game means: keep the currently selected dictionary, clear the entered guess history, clear the feedback editor, and return the solver to a fresh puzzle state.

Changing the dictionary is a separate action. If you switch from bundled to cached remote, or activate a different cached remote dictionary, you are changing the solver’s word bank as well as starting from a fresh puzzle state.

So in practical terms: if you are starting the next daily Wordle with the same word list, click New Game. If you want to analyze with a different word list, change dictionaries instead.

Engine Status Bar

The status bar above the workspace reports what the engine is doing right now.

  • Idle in green means the solver is ready for another command.
  • Working in yellow means the app is fetching a dictionary, rebuilding solver state, or recomputing recommendations.
  • Error in red means the last operation failed and the detail text explains what went wrong.

If the solver feels latent on a larger dictionary, watch the status bar first. That is the intended place to tell whether the app is still processing versus waiting for user input.

Privacy

Wordle Wizard does not collect analytics, does not send your guesses or puzzle history to any server, and does not store personal data remotely.

The only persistent storage used by the app is your browser's local storage for cached dictionaries that you explicitly fetch. Those cached dictionaries stay on your device unless you remove them or clear browser storage.
